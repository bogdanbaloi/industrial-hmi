# Application Logs

This directory contains runtime application logs.

Log files:
- app.log - Main application log
- errors.log - Error-level and above

Log format:
[TIMESTAMP] [LEVEL] [FILE:LINE] MESSAGE

Example:
[2026-04-09 10:45:23.123] [INFO ] [MainWindow.cpp:15] Application starting
[2026-04-09 10:45:23.456] [ERROR] [DatabaseManager.cpp:89] Connection failed
